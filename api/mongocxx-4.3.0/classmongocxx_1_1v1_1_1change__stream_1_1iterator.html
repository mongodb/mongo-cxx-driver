mongocxx::v1::change_stream::iterator Class Reference

#include <mongocxx/v1/change_stream.hpp>

Description

An iterator over change stream events.

Attention
This feature is experimental! It is not ready for use!
Important
All iterators associated with the same change stream object share the same state.
The end iterator has no associated change stream.
Note
This iterator almost satisfies Cpp17ForwardIterator, but std::iterator_traits<T>::reference is defined as value_type, similar to std::vector<bool>::iterator and std::istreambuf_iterator<T>. Therefore, this iterator only fully satisfies Cpp17InputIterator.
