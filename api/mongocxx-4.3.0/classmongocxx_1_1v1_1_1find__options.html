mongocxx::v1::find_options Class Reference

#include <mongocxx/v1/find_options.hpp>

Description

Options for a "find" command.

Supported fields include:

  • allow_disk_use ("allowDiskUse")
  • allow_partial_results ("allowPartialResults")
  • batch_size ("batchSize")
  • collation
  • comment
  • cursor_type ("cursorType")
  • hint
  • let
  • limit
  • max_await_time ("maxAwaitTimeMS")
  • max_time ("maxTimeMS")
  • max
  • min
  • no_cursor_timeout ("noCursorTimeout")
  • projection
  • read_preference ("readPreference")
  • read_concern ("readConcern")
  • return_key ("returnKey")
  • show_record_id ("showRecordId")
  • skip
  • sort
See also
  • find (database command) (MongoDB Manual)
Attention
