mongocxx::v1::delete_one_options Class Reference

#include <mongocxx/v1/delete_one_options.hpp>

Description

Options for a "deleteOne" operation.

Supported fields include:

  • collation
  • hint
  • let
  • write_concern ("writeConcern")
  • comment
See also
  • Delete Methods (MongoDB Manual)
