mongocxx::v1::write_concern Class Reference

#include <mongocxx/v1/write_concern.hpp>

Description

Options related to a MongoDB Write Concern.

Supported fields include:

  • journal ("j")
  • level ("w")
  • timeout ("wtimeout")
See also
  • Write Concern (MongoDB Manual)
Attention
This feature is experimental! It is not ready for use!

◆ acknowledge_level() [1/2]

level mongocxx::v1::write_concern::acknowledge_level ( ) const

Return the current "w" field as an acknowledgement level.

Returns
  • level::k_default when "w" is unset.
  • level::k_majority when "w": "majority".
  • level::k_tag when "w": <custom write concern name>.
  • level::k_unacknowledged when "w": 0.
  • level::k_acknowledged when "w": n and n >= 1.
  • level::k_unknown for all other (unsupported) values.
