mongocxx::v1::range_options Class Reference

#include <mongocxx/v1/range_options.hpp>

Description

Options related to range_options queries for Queryable Encryption.

Supported fields include:

  • max
  • min
  • precision
  • sparsity
  • trim_factor ("trimFactor")
See also
  • Client Side Encryption (MongoDB Specifications)
  • Encrypted Fields and Enabled Queries (MongoDB Manual)
Attention
This feature is experimental! It is not ready for use!
