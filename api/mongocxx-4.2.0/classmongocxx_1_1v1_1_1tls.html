mongocxx::v1::tls Class Reference

#include <mongocxx/v1/tls.hpp>

Description

Options related to TLS configuration.

Supported fields include:

  • allow_invalid_certificates ("weak_cert_validation")
  • ca_dir
  • ca_file
  • crl_file
  • pem_file
  • pem_password ("pem_pwd")
See also
  • mongoc_ssl_opt_t (mongoc)
  • TLS/SSL (Transport Encryption) (MongoDB Manual)
