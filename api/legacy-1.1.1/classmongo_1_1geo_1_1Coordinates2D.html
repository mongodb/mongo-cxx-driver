Detailed Description

Represents a 2D position (x, y).

Example Usage:

Coordinates2D coords(1, 2); Point<Coordinates2D> point(coords); Query nearQuery = QUERY("<field_name>" << NEAR(point)); conn.query("<db_name>.<collection_name>", nearQuery);
