#include <mongocxx/v1/count_options.hpp>

Description

Options for a "countDocuments" operation.

Supported fields include:

  • collation
  • comment
  • hint
  • limit
  • max_time ("maxTimeMS")
  • read_concern ("readConcern")
  • read_preference ("readPreference")
  • skip
